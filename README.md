# Search Tool Chrome Extension

A simple Chrome extension that provides a quick way to search using different search engines.

## Features

*   **Quick Access:** Open the search popup with the `Ctrl+I` shortcut.
*   **Multiple Search Engines:** Easily switch between DuckDuckGo, Google, Bing, and Yandex.
*   **Keyboard Navigation:**
    *   The search input is focused automatically.
    *   Use number keys `1` through `4` to select your desired search engine.
    *   Press `Enter` to execute the search.
*   **Clean UI:** A simple and intuitive interface.

## How to Install (Unpacked)

1.  Open Chrome and navigate to `chrome://extensions`.
2.  Enable "Developer mode" in the top right corner.
3.  Click on "Load unpacked".
4.  Select the directory where you have cloned or downloaded this project.
5.  The extension should now be installed and ready to use. You will see its icon in the Chrome toolbar.
